Интернет Магазин написанный на <b>Python</b> \
С использованием библеотек <b>Django</b>, <b>Celery</b> \
И базы данных <b>PostgreSQL</b>

<h3>Функционал:</h3>

- Мультиязычность
- Спам рассылка (Celery)
- Админка 
- Оплата (Stripe)
- Пагинация
- и т.д
